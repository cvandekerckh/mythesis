# MyThesis

Static HTML pages used as QR-code destinations in my master's thesis. Each page is a short landing page linked from a printed QR code in the manuscript.

The thesis is available on UCLouvain's institutional repository (DIAL):
https://dial.uclouvain.be/pr/boreal/object/boreal:188323

## Contents
The repository hosts one HTML page per region/section referenced in the thesis (e.g. `quebec.html`, `wallonie.html`, `flandre.html`, `canada.html`, `new-zealand.html`), served via GitHub Pages.

## Status
Archived companion material for a completed thesis (2021).
